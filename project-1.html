Server

This is a Synology DS220+ Server that I am using as my personal cloud storage, media storage and streaming station, and most importantly
as a web hosting platform.

Project Overview

There is an initial investment required for my server setup of a couple hundred dollars,
but it is a great investment for people who want to get into the networking/server
environment. I personally did not know anything about servers and/or how they work
until I got my hands on one of these. To get to the level that I have gotten it to, many things are required such as knowledge about local and public ip's, how hostnames work, and how to work around the terminal/command-prompt.

To learn, I watched videos from NasCompares's and SpaceRex's Youtube channels, and read through many of Marius Hosting's detailed websites explaining different things that can be done with my Server.

I was also thinking of making my own rack server, but that gets too expensive too quickly, and if you do there are many videos and articles out there that can help along the way, and if you have any questions please message me from the contact page, I would love to help out.
